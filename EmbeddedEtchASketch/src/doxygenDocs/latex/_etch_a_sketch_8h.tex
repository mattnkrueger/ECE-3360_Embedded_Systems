Function that handles commands relating to the Etch A Sketch. 

Color cycling with arrows and RPG movement for drawing.
\begin{DoxyItemize}
\item Up arrow increments the pointer of the color
\item Down arrow decrements the pointer
\item RPG2 (left) controls left/right (ccw/cw)
\item RPG1 (right) controls up/down (ccw/cw)
